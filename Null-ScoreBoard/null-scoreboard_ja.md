# Japanese

## Null - Score Board&#x20;

モダンでレスポンシブなデザインを持つ、QBCoreフレームワーク向けの多機能スコアボードです。サーバーの情報、実行可能な犯罪リスト、そしてあなた自身のプレイヤー情報を、見やすくスタイリッシュなUIで表示します。 設定ファイルから簡単にカスタマイズでき、多言語にも対応しています。

***

### ✨ 主な機能 (Main Features)

* **サーバー情報表示**: オンラインのプレイヤー数、警察官、救急隊の人数を表示します。
* **あなた自身の情報表示**: あなたのID、名前、Pingを個別に表示します。
* **犯罪リスト**: `config.lua`で設定した犯罪を一覧表示します。
  * 実行に必要な警察官の人数に応じて、ステータス（可能/不可）がリアルタイムに変化します。
  * リストの項目数に応じて、UIの横幅と列数が自動で調整されるレスポンシブデザイン。
* **自動バージョンチェック**: サーバー起動時に新しいアップデートがないかを自動で確認し、コンソールに通知します。
* **完全なカスタマイズ性**: `config.lua`から、ほぼ全ての表示項目を個別にON/OFFできます。
* **柔軟なソート機能**: 犯罪リストを「必要警察官の少ない順」または「多い順」に並び替え可能です。
* **多言語対応**: `locales`フォルダに言語ファイルを追加するだけで、簡単に新しい言語に対応させることができます。（デフォルトで日本語と英語が付属）
* **キー設定**: プレイヤーはゲーム内のキー設定メニューから、スコアボードを開くキーを自由に変更できます。
* **軽量設計**: 待機時のリソース消費は**0.00ms**で、サーバーのパフォーマンスに影響を与えません。

### 📥 導入方法 (Installation)

1. 購入後、CFX Keymaster Portalから`null-scoreboard`のzipファイルをダウンロードします。
2. ダウンロードしたzipファイルを解凍し、`null-scoreboard`フォルダをあなたのサーバーの`resources`ディレクトリに配置します。
3.  `server.cfg`ファイルを開き、以下の行を追記します。

    ```cfg
    ensure null-scoreboard
    ```
4. サーバーを再起動すれば、導入は完了です。

***

### ⚙️ 設定方法 (Configuration)

設定はすべて`config.lua`と`locales`フォルダ内のファイルで行います。

#### `config.lua`

```lua
Config = {}

-- 表示したい言語を設定 ('ja' or 'en')
Config.Lang = 'ja'

-- プレイヤーがキー設定で何も設定していない場合の、デフォルトのキー
Config.DefaultKey = 'Home'

-- 犯罪リストの並び順を設定
-- 'asc': 必要警察官が少ない順
-- 'desc': 必要警察官が多い順
-- false: このファイルに記述した通りの順番
Config.SortCrimeList = 'asc'

-- QBCoreでの実際のジョブ名を指定
Config.Job = {
    police = 'police',  
    ambulance = 'ambulance'
}

-- UIの各項目の表示/非表示を切り替える (true or false)
Config.Display = {
    player = true,      -- サーバー全体のプレイヤー数
    police = true,      -- 警察官の数
    ambulance = true,   -- 救急隊の数
    id = true,          -- あなた自身のID
    name = true,        -- あなた自身の名前
    ping = true,        -- あなた自身のPing
}

-- 表示したい犯罪のリスト
Config.crimelist = {
    ['storerobbery'] = {       -- 犯罪のユニークなID
        label = "コンビニ強盗", -- UIに表示される名前
        minpolice = 2,         -- 実行に必要な最低警察官の数
        status = false         -- 常にfalseのままにしてください
    },
    -- ...
}

### locales/ja.lua (言語ファイル)

UIに表示される全てのテキストは、localesフォルダ内の言語ファイルで編集できます。新しい言語を追加したい場合は、ja.luaをコピーしてes.lua（スペイン語）などを作成し、中のテキストを翻訳してください。
```
